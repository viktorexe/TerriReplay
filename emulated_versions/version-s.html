Hovering Tooltip

Adds a tooltip in the bottom right corner when hovering
or clicking on a territory, showing some extra information

Tooltip Formats

The following arguments which get replaced when displayed:
<name> - player name
<troops> - troop count
<territory> - territory size
<defence> - border defence
<arrival> - boat arrival time (boat only)
<boat> - boat troop count (boat only)
